# -tempjqlate
这是一个jq的模板仓库，技术栈：webpack + jq + less + eslint 
##项目目录
```
.
├── build ---------------------- webpack配置文件
├── src ------------------------- 源码目录
    ├── index.js ---------------- js脚本  
    ├── index.less ------------------ less样式文件
├── babelrc ----------------------------- babel配置
├── .eslintrc.js ------------------------ eslint配置
├── .gitignore -------------------------- git忽略文件配置
├── index.html -------------------------- 模板html文件
└── package.json ------------------------ 都懂的
├── postcss.config.js ------------------- postcss配置
├── webpack.config.js ------------------- 判断环境
```
